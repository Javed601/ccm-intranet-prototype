Button Styles

  • .btn-primary: Main action, blue background, white text
  • .btn-secondary: Neutral action, light gray background, blue text
  • .btn-outline: Transparent, blue border, blue text

Other Elements to Consider

  • Navigation bar (top or side)
  • Cards and panels for content
  • Consistent icon set (e.g., Fluent UI, FontAwesome)
  • Forms (inputs, selects, checkboxes, etc.)
  • Alerts and notifications
  • Tables and lists
  • Responsive layout (mobile-friendly)
  • Accessibility (contrast, keyboard navigation)

SharePoint Considerations

  • Use web-safe fonts (Segoe UI, Arial)
  • Keep CSS simple and avoid !important
  • Use SharePoint-compatible HTML (no custom scripts unless allowed)
  • Test in SharePoint Modern Pages
  • Use Office UI Fabric/Fluent UI for advanced components
  • Ensure accessibility and responsive design
  • Minimize external dependencies
